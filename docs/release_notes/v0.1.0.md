# Dapr 0.1.0

This is this initial release of Dapr.  In this version Dapr provides the following building blocks:

  1. [Messaging](https://github.com/dapr/docs/blob/master/reference/api/service_invocation.md)
  2. [Bindings](https://github.com/dapr/docs/blob/master/reference/api/bindings.md)
  3. [Pub Sub/Broadcast](https://github.com/dapr/docs/blob/master/reference/api/pubsub.md)
  4. [State Management](https://github.com/dapr/docs/blob/master/reference/api/state.md)
  5. [Actors](https://github.com/dapr/docs/blob/master/reference/api/actors.md)
  6. [Metadata](https://github.com/dapr/docs/blob/master/reference/api/metadata.md)
